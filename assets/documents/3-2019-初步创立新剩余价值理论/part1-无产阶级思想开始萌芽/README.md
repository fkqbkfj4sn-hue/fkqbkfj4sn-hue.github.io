# Part 1：无产阶级思想开始萌芽（四篇文章）

## 思想定位
这四篇文章代表了我向马克思主义立场和方法论的初步转向，尝试用现代科学方法重新解读和发展马克思主义政治经济学。

## 四篇文章的思想递进

### 1. 无差别人类劳动的算法和实证
- **核心问题**：如何用数学方法刻画马克思的"抽象劳动"概念
- **方法论**：建立劳动同质化的算法模型
- **实证尝试**：用实际数据检验劳动价值论基础
- **思想意义**：开始摆脱西方经济学的效用价值论

### 2. 商品二重性的算法和实证
- **核心问题**：商品使用价值与价值的数学表达
- **理论贡献**：将商品二重性转化为可计算的模型
- **实证设计**：设计商品二重性的检验方法
- **思想意义**：深化对马克思主义方法论的认同

### 3. 资本的算法和实证
- **核心问题**：资本运动规律的数学刻画
- **模型创新**：建立资本循环的算法模型
- **实证分析**：用中国数据验证资本运动规律
- **思想意义**：开始形成马克思主义资本分析框架

### 4. 中国基本经济制度的自生发展
- **核心问题**：中国特色社会主义经济制度的理论阐释
- **理论视角**：从马克思主义立场分析中国制度
- **实践意义**：为中国特色社会主义提供理论支持
- **思想意义**：确立马克思主义中国化的研究方向

## 总体思想特征
1. **方法论自觉**：明确采用马克思主义分析方法
2. **科学化尝试**：将定性理论转化为定量模型
3. **中国化探索**：将马克思主义与中国实践结合
4. **批判性继承**：在批判西方经济学基础上创新
