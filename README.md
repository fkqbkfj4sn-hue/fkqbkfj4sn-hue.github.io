# 历史研究成果存档 - 新剩余价值理论发展史

## 仓库说明
本仓库系统化存档了从2007年至2019年的学术研究成果，真实记录了从具体经济现象研究到新剩余价值理论初步创立的思想发展历程，特别呈现了理论创新过程中的思想双重性和转变轨迹。

## 思想发展脉络

### 1. 2007年：实证研究的起点
- **研究**：尝试解析农民经济组织
- **方法**：内生性分析框架
- **定位**：从具体经济现象出发的实证研究

### 2. 2015年：经典理论的学习
- **研究**：初步接触《资本论》中马克思建立的链模型
- **方法**：文本分析与逻辑推演
- **特征**：论文录用但未现场参会，标志理论学习阶段

### 3. 2019年：理论创新的尝试与思想双重性
- **总体**：初步创立新剩余价值理论
- **Part 1**：无产阶级思想开始萌芽（四篇文章）
  - 用算法和实证方法探索马克思主义核心概念
  - 体现向马克思主义方法论的初步转向
- **Part 2**：非无产阶级思想仍旧存在（一篇文章）
  - 反映西方经济学分析框架的残留影响
  - 呈现理论转型期的思想矛盾性

## 学术诚实与反思
本存档特别注重呈现真实的思想发展过程，包括：
1. 理论立场转变的不彻底性
2. 方法论选择的矛盾性
3. 概念工具的混杂性
4. 自我批判和理论清算的尝试
