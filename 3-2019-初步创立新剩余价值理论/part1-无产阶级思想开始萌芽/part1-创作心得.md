# Part 1：四篇论文创作心得

## 总体思想背景
2019年是我思想转变的关键时期，开始尝试用马克思主义立场和方法论分析经济问题。这四篇论文体现了我向无产阶级思想方法的初步转向。

## 各篇核心思考
### 1. 无差别人类劳动的算法和实证
- 尝试用算法表达马克思的抽象劳动概念
- 探索劳动价值论的现代数学基础

### 2. 商品二重性的算法和实证
- 商品使用价值与价值的数理表达
- 尝试建立商品二重性的检验方法

### 3. 资本的算法和实证
- 资本运动规律的数学刻画
- 用中国数据验证理论模型

### 4. 中国基本经济制度的自生发展
- 从马克思主义立场分析中国制度
- 探索中国特色社会主义的理论基础

## 方法论反思
这四篇文章共同特点是：尝试用现代科学方法（算法、实证）重新表达和发展马克思主义经济学。
