# 2019年研究：初步创立新剩余价值理论

## 思想发展背景
2019年是中国政治经济学年会收录五篇论文的关键年份，标志着我从学习经典理论到尝试创新理论的转变。这个时期思想呈现明显的双重性。

## 思想结构分析

### Part 1：无产阶级思想开始萌芽
四篇论文体现马克思主义立场和方法论的初步确立：
1. **方法论突破**：尝试用算法和实证方法分析马克思主义核心概念
2. **理论创新**：对劳动、商品、资本进行数学重构
3. **中国视角**：从中国实践出发理解基本经济制度

### Part 2：非无产阶级思想仍旧存在
一篇论文反映思想中仍存在的传统经济学影响：
- **分析框架**：受西方经济学有机体理论影响
- **理论局限**：未能完全摆脱非马克思主义范式
- **思想张力**：体现理论创新过程中的内在矛盾

## 学术自觉
这种思想双重性真实反映了理论创新初期的探索特征：既有突破传统的勇气，又难以完全摆脱既有理论范式的影响。
